# Moltis Intelligence

A standalone AI agent that operates within GitHub's native infrastructure.

Powered by the [Moltis](https://github.com/moltis-org/moltis) runtime — a single Rust binary with zero `unsafe` code, sandboxed execution, built-in memory, and multi-provider LLM support.

Every conversation lives in a GitHub Issue. Every response is committed to Git. Every secret stays in GitHub Secrets. The repository is the mind.
